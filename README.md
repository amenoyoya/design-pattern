# デザインパターン入門

## Introduction

結城浩・著「Java言語で学ぶデザインパターン入門」をTypeScriptで演習

***

## Environment

- OS:
    - Ubuntu 18.04
    - Windows 10
- Jupyter Notebook: `4.4.0`
    - Anaconda: `4.5.11`
        - Python: `3.7.4`
- NodeJS: `10.16.2`
    - yarn: `1.17.3`

### Setup

```bash
# install TypeScript Julia kernel: ITypeScript
$ yarn global add itypescript
$ export PATH="$PATH:$(yarn global bin)"
$ its --install=local
```
